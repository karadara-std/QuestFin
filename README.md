# 🎮 FinQuest — Геймифицированные личные финансы

> Приложение, где твои финансы превращаются в RPG. Твой персонаж растёт вместе с твоей финансовой грамотностью.

## 🌟 Проблема

85% людей не ведут бюджет — потому что это скучно. Дети не понимают ценность денег. Малый бизнес тонет в кассовых разрывах. Существующие финансовые приложения — это таблицы с числами. FinQuest превращает финансовую дисциплину в привычку через эмоциональную связь с персонажем.

---

## 🎯 Решение

**Три режима, одна платформа:**

| Режим | Для кого | Ключевая механика |
|-------|----------|-------------------|
| 🧑 Личный | Все пользователи | Персонаж эволюционирует с уровнем финграмотности |
| 👨‍👩‍👧 Семейный | Родители + дети | Квесты от родителей, семейные цели, общий прогресс |
| 🏪 Бизнес | МСБ-предприниматели | Прогноз кассового разрыва за 14 дней |
